== WooCommerce Remove Product Sorting ==
Author: skyverge
Tags: woocommerce, sorting
Requires at least: 4.4
Tested up to: 4.9.6

Adds the ability to remove core WooCommerce sorting options from your shop.

See full details [on SkyVerge.com](http://skyverge.com/product/woocommerce-remove-product-sorting/).

== Installation ==

1. Upload the entire 'woocommerce-remove-product-sorting' folder to the '/wp-content/plugins/' directory, or upload the zip file under Plugins > Add New > Upload
2. Activate the plugin through the 'Plugins' menu in WordPress
